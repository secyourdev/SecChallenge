FOREMOST : LOGICIEL DE RÉCUPÉRATION DE DONNÉES


1. Introduction

Foremost est un logiciel Linux de récupération de données pour l'informatique légale, qui exploite les en-têtes, pieds de page et structures de données de fichiers endommagés via un processus nommé file carving, consistant en la récupération à partir de fragments privés de métadonnées de fichiers informatiques.
En effet, Foremost est capable de travailler sur des fichiers image ou même directement sur un disque dur, car il permet l'utilisation de modificateurs de ligne de commande pour spécifier les types de fichiers que nous voulons rechercher et ainsi être plus précis avec ce que nous souhaitons rechercher.

Ce logiciel donne la possibilité de récupérer des fichiers sous une multitude de formats, comme jpg, gif, png, bmp, avi, tiff, mp4, exe, mpg, wav, asf, wma, mp3, fws, riff, wmv, mov, pdf, ole, doc, docx, xls, xlsx. ppt, pptx, zip, rar, html, cpp, java, art, pst, ost, dbx, idx, mbx, wpc, pgp, txt, rpm, dat, etc.


2. Installation de Foremost

Foremost est un outil en ligne de commande, disponible dans les dépôts officiels de Kali Linux.
Pour l’installer, il suffit ensuite de taper la commande suivante :
sudo apt install foremost


3. Fonctionnement

Pour comprendre comment Foremost fonctionne et ce que le logiciel nous permet de réaliser, vous pouvez taper la commande ci-dessous :
foremost -h

On peut alors observer toutes les options à notre disposition :

  • -V : affiche l'information sur le droit d'auteur puis quitte ;

  • -t : permet de spécifier le type de fichier (ex : jpeg, pdf);

  • -d : permet l'activation de la détection indirecte des blocs (pour les systèmes de fichiers UNIX);

  • -i : spécifie le fichier d'entrée (stdin par défaut);

  • -a : permet l'écriture de tous les en-têtes, avec aucune détection d'erreur (fichiers corrompus);

  • -w : permet d'écrire uniquement à l'intérieur du fichier d'audit, et non dans les autres fichiers détectés sur le disque ;

  • -o : sert à définir le répertoire de sortie du fichier ;

  • -c : définit le fichier de configuration à utiliser (foremost.conf par défaut);

  • -q : active le mode rapide, les recherches sont alors effectuées sur des frontières de 512 octets.;

  • -Q : active le mode silencieux et supprime les messages de sortie ;

  • -v : mode verbeux, enregistre tous les messages à l'écran;


4. Exemples

Dans cette partie, vous pourrez voir des commandes concrètes qui peuvent s'utiliser en cas réels, afin de vous aider à mieux visualiser le fonctionnement de Foremost :

  • foremost -s 100 -t jpg -i image.dd : recherche le format jpeg en sautant les 100 premiers blocs ;
  • foremost -av image.dd : génère uniquement un fichier d'audit, et l'affiche à l'écran (mode verbeux);
  • foremost -t all -i image.dd : recherche tous les types définis ;
  • foremost -t gif,pdf -i image.dd : recherche les formats gif et pdf;
  • foremost -vd -t ole,jpeg -i image.dd : permet la recherche de documents office et de fichiers jpeg dans un système de fichiers Unix en mode verbeux ;
  • foremost image.dd : exécute le cas par défaut ;
Pour voir une mise en pratique avec illustration, vous pouvez aussi jeter un coup d'œil sur ce site par exemple, qui montre une récupération de fichiers en plusieurs étapes.
