Cours brut-force - Ncrack

Introduction

Ncrack est un outil de craquage de d'indentifiant/mot de passe en réseau puissant et flexible développé dans le cadre du projet Nmap. Il peut vous aider à tester la sécurité de votre réseau, à identifier les identifiants de connexion faibles.

Installation

Ncrack est disponible pour Linux, Windows et macOS. Voici comment l'installer sur chaque plateforme :

  • Linux : La plupart des distributions Linux ont Ncrack disponible dans leurs dépôts de paquets. Vous pouvez l'installer en utilisant le gestionnaire de paquets de votre distribution (par exemple, apt, yum ou pacman).
  • Windows : Téléchargez la dernière version de Ncrack pour Windows à partir du site officiel (https://nmap.org/ncrack/), et extrayez l'archive ZIP. Ajoutez le dossier extrait à la variable PATH de votre système.
  • macOS : Vous pouvez installer Ncrack en utilisant Homebrew (https://brew.sh/). Lancez la commande brew install ncrack dans le terminal.
  • docker : Récupérez le dockerfile dans le lien suivant, si vous avez déjà installer le docker-compose.yml du challenge ou du tutoriel pas la peine de le reprendre docker-compose.yml
Dans un interpréteur de commande,bash (Linux) ou cmd (Windows), allez dans le répertoire où vous avez téléchargé le fichier docker-compose.yml et effectuez la commande suivante :
docker compose up -d

Vous disposez maintenant d'une machine avec Spider Foot. Pour mettre en pause ou arreter la machine faite:
docker compose pause/unpause
docker compose stop/start

Si vous voulez l'installer par vous-même, il est téléchargeable depuis le site officiel : SpiderFoot.

Utilisation basique

La syntaxe de base de Ncrack est la suivante :

    ncrack [options] {cible}:{service}:[port] [login]

options : n'importe quel flag optionel pour modifier le comportement de Ncrack.
cible : L'adresse IP du système cible.
service : Le nom du service que vous voulez craquer (par exemple, ssh, ftp, http).
port : Le numéro de port pour le service (facultatif).
login : La source des identifiants (par exemple, un fichier de nom d'utilisateur et de mot de passe).

Exemples :

  • Forcer une connexion SSH :
    • ncrack -p 22 -U root -P 'file:passwords.txt' ssh://192.168.1.1
  • Craquer une connexion FTP en utilisant une liste de noms d'utilisateur et de mots de passe :
    • ncrack -p 21 -U 'file:usernames.txt' -pass password,qwerty,123 ftp://192.168.1.1

Options avancées

Voici quelques options avancées qui peuvent être utiles lors de l'utilisation de Ncrack :

  • -T <modèle de temporisation> : Spécifiez un modèle de temporisation (0-5) pour ajuster la vitesse du processus de craquage. Les valeurs plus élevées sont plus rapides mais plus agressives, tandis que les valeurs plus faibles sont plus lentes mais moins susceptibles d'être détectées.
  • --connection-limit <num> : Définir le nombre maximal de connexions simultanées.
  • --proxy <type://[user:pass@]host:port> : Utilisez un serveur proxy pour les connexions.
  • --resume <file> : Reprendre une session Ncrack précédemment enregistrée.

Conseils pour un craquage efficace

  • Utilisez de grandes listes de mots de passe diversifiées pour augmenter les chances de découvrir des informations d'identification faibles.
  • Tenez compte des mesures de sécurité du système cible, telles que les verrouillages de compte et la limitation de la vitesse, lors du choix d'un modèle de temporisation.
  • Soyez conscient des implications légales et éthiques lors de l'utilisation de Ncrack sur des systèmes que vous ne possédez pas ou pour lesquels vous n'avez pas la permission de tester.

Conclusion

Ncrack est un outil puissant de craquage d'authentification de réseau qui peut vous aider à identifier des informations d'identification faibles et à améliorer la sécurité de vos systèmes. En comprenant son utilisation de base et avancée, vous pouvez tester et sécuriser efficacement votre réseau contre l'accès non autorisé.
