NetScanTools

1. Qu'est-ce que NetScanTools ?

NetScanTools est un logiciel de réseau développé par Northwest Performance Software.
Il regroupe un ensemble d'outils conçus pour l'analyse et la gestion des réseaux informatiques.

2. Comment installer NetScanTools ?

Il faut tout d'abord installer la démo du logiciel NetScanTools Pro. La période d'essai est de 30 jours.
Le lien pour la télécharger est ici:

3. Applications les plus utiles en cybersécurité

  • L'outil automatique Enter and Retrieve Data

    Cet outil permet de lancer une batterie de tests en même temps sur une cible qui peut être une adresse IP, un nom de domaine, une adresse email ou encore un URL.
    Le rapport est enfin accessible dans l'outil View automated mode reports.

  • L'outil Whois

    Cet outil permet de récupérer des informations sur un nom de domaine ou une adresse IP à partir de la base de données WHOIS.
    Cette base de données contient des informations sur l'enregistrement et la propriété des noms de domaine et des adresses IP.
    Exemple avec le nom de domaine microsoft.com:

  • L'outil PING

    Cet outil permet de tester l'accessibilité et la réactivité d'un hôte du réseau ou d'une adresse IP.
    Pour des adresses IPv4, il est possible d'envoyer des messages ICMP, UDP, TCP, UDP Socket Xmit et MS ICMP.
    Trois ping utilisent WinPcap pour avoir une réponse plus rapide: ICMP, UDP et TCP.
    Exemple pour pinger le site de Google:

  • L'outil TCP Traceroute

    Cet outil permet d'envoyer des messages TCP à l'adresse cible et renvoie des messages ICMP et des messages TCP en utilisant WinPcap.
    L'outil montre le trajet parcouru par les messages de la machine host vers l'adresse cible.
    Exemple avec le site de Google:

  • L'outil ARP Scan (scan MAC)

    Cet outil permet de découvrir et mapper les adresses MAC (Media Access Control) associées aux adresses IP sur un réseau local.
    L'outil fonctionne d'abord en lançant des requêtes ARP à toutes les adresses IP d'une plage spécifiée sur le réseau local.
    Ensuite, il attend les réponses des appareils du réseau qui possèdent ces adresses IP et fait une correspondance adresse IP/MAC.
    La table MAC est ensuite renvoyée à l'utilisateur.

  • L'outil DNS Tools CORE

    Cet outil est un ensemble d'utilitaires pouvant effectuer différentes tâches liées au système de noms de domaine (DNS).
    Le DNS est un protocole fondamental utilisé sur l'internet pour traduire les noms de domaine lisibles par l'homme en adresse IP que les ordinateurs peuvent comprendre.

    • Who am I?

      • Renvoie les informations WHOIS de ta machine.

    • DNS Lookup et DNS Lookup

      • Récupère des informations sur un nom de domaine et inversement en utilisant son adresse IPv4.
