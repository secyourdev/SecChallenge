Network Topology Mapper

1. Qu'est-ce que Network Topology Mapper ?

Network Topology Mapper (NTM) est un logiciel développé par SolarWinds. Il permet d'effectuer des scans réseaux et réaliser une vue d'ensemble topologique de son réseau.

2. Comment Network Topology Mapper fonctionne ?

NTM utilise les protocoles suivants pour découvrir les noeuds du réseau et la connectivité entre eux :

  • Simple Network Management Protocol (SNMP)
  • Windows Management Instrumentation (WMI)
  • Link Layer Discovery Protocol (LLDP)
  • Cisco Discovery Protocol
  • VMware Management
  • Internet Control Message Protocol (ICMP ping)

    4. Comment utilise-t-on Network Topology Mapper ?

    • Créer une map

      Pour créer une map, tu dois lancer un scan. Si tu vienx de lancer l'application une fenêtre t'invitera à lancer un nouveau scan.
      Ajoute ensuite les informations d'identification SNMP utilisées pour la découverte et le stockage des appareils réseaux.




      Dans la fenêtre "Network selection" renseigne les parties du réseau que tu souhaites scanner. Plusieurs possibilités :
      • Ajouter tout un sous réseau avec son adresse et son masque
      • Ajouter les machines connectés à l'IP donnée
      • Donner une plage d'adresses
