Traceroute

1. Objectif

L'objectif de ce tutoriel est de vous présenter l'outil de ligne de commande traceroute.

2. Configuration

Ce tuto met à disposition, par le biais de l'application Docker et docker-compose, trois machines connectées sur un même réseau local. Les commandes qui permettent de lancer les 3 machines sont les suivantes :
bash> docker-compose up -d

Nous avons besoin de nous connecter à celle d'Alice via la commande :
bash> docker exec -it reseau1_alice /bin/bash

On doit maintenant définir les routes de routage. Attention a bien les parametrer dans les deux sens. bash> route add default gw 192.168.111.3 eth0

Il faut ensuite installer traceroute bash> sudo apt-get install traceroute
