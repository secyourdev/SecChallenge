Tutoriel : Proxy Switcher

1. Proxy Switcher 101

Lorsque plusieurs proxy sont en jeu, généralement pour des usages différents, l’utilisateur peut avoir besoin de changer régulièrement de proxy.
Ainsi Proxy Switcher sert à simplifier plusieurs éléments tels que le passage de l’utilisation d’un proxy à celle d’un autre, la configuration des proxy, la gestion des profils dans les proxy, ou même leurs performances.

2. Utilisation

Proxy Switcher est un outil à utiliser en parallèle d'un navigateur web.
Il permet d'exploiter au mieux les avantages de différents proxy.

3. Confidentialité

Comme les proxy effectuent les requêtes pour le client, l'adresse IP du client n'apparait pas, à première vue du moins, et celle du proxy en cours d'utilisation prend sa place.
L'anonymité du client est donc renforcée.

4. Optimisation des performances

Lors de l'envoi de requêtes, les proxy peuvent stocker des données en cache et les réutiliser pour des requêtes ultérieures.
Comme le client a accès à plusieurs proxy, il peut décider de les utiliser parallèlement pour effectuer plusieurs tâches.

5. Localisation

Les proxy ne sont pas toujours situé à proximité du client, ainsi choisir un proxy localisé dans un pays différent, donc avec des restrictions différentes, donne au client une adresse IP dans ce pays.
Le client peut donc accéder à des contenus différents.
