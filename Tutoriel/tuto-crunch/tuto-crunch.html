Tutoriel : crunch

1. Crunch 101

Crunch est un outil permettant de générer des listes de mots, ou plus généralement de caractères.
Il peut générer ces listes à partir de lignes de commandes spécifiant le nombre de caractères, les caractères à utiliser et même certains paternes de ces caractères.

2. A quoi ça sert tout ça ?

Dans le contexte de la cybersécurité, les listes générées par crunch seront souvent des mots de passe, que ce soit pour choisir un mot de passe pour un site quelconque ou pour avoir une liste de mots de passe potentiels pour faire du bruteforce et gagner ainsi l'accès à une machine ou à des données protégées.

3. Configuration et execution

Récupérer le dockerfile dans le lien ci-après : docker-compose.yml

Lancer le docker avec :
bash> docker compose up -d

Accéder à l’invite de commande avec :
bash> docker exec -it tuto_xxx-syd-crunch bash

On va créer une liste de mots de passe de 3 caractères dont les caractères sont “azerty” avec:
bash> crunch 3 3 azerty -o liste


On a donc une liste de mots de passe dans un fichier liste. Pour vérifier on va afficher la ligne 36 du fichier avec:
bash> sed -n '36 p' ./liste


Pour quitter:
bash> exit
bash> docker compose down

4. Validation

Pour valider le tutoriel, indiquez le contenu de la ligne 36.
