HTTrack

1. Objectif de l'outil

HTTrack est un aspirateur de sites web. Cet outil permet de télécharger l'intégralité d'un site Internet directement sur le disque dur de l'utilisateur, en reconstruisant tous les répertoires. C'est particulièrement utile pour consulter un site en mode hors ligne ou pour archiver des informations.
HTTrack est capable de mettre à jour une copie existante d'un site Web et de reprendre les téléchargements interrompus. Il est entièrement configurable, et possède un système intégré d'aide.

2. Configuration

Tout d'abord veuller installer le fichier docker ici présent ainsi que le site à copier. Pour utiliser HTTrack, vous pouvez le télécharger et l'installer à partir du site officiel et ensuite suivre les indication du logiciel ou utiliser les gestionnaires de paquets des distributions Linux.
Tapez la commande suivante pour installer HTTrack sur Ubuntu :

sudo apt-get install httrack

Une fois installé, vous pouvez lancer HTTrack à partir de la ligne de commande, ou utiliser WebHTTrack, une interface Web pour HTTrack.

3. Utilisation de HTTrack

Tout d'abord veullez telecharger ce fichier docker et cette page html.

Après avoir télécharger les deux fichier, vous devez mettre les deux fichier dans le même dossier et metre le index.html dans un sousdossier nommé 'html'.
Puis lancer votre cmd aller dans le repertoire ou se trouve les fichier télécharger et lancer le docker-compose avec la commande :
docker-compose up -d

Après cela aller sur votre moteur de recherche et taper : http://localhost/
Vous pouvez maintenant lancer HTTRACK et lancer la copie du site web en suivant les étape indiquer sur le logiciel. Vous pourrez ensuite avoir accès au coe du site web dans le répertoire ou vou avez telecharger le site.
