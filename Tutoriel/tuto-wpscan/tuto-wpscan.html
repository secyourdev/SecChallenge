WPScan

1. Objectif de l'outil

Nous savons que près de 50% des sites sur Internet sont hébergés par WordPress. Donc, s'il existe un scanner de vulnérabilité spécifiquement pour WordPress, c'est génial.

L'outil WPScan CLI est un scanner de sécurité WordPress gratuit afin de tester la sécurité des sites WoldPress.

2. Configuration

Pour ce tutoriel, nous utiliserons docker pour avoir une machine pour lancer le site web WorldPress et une machine attaquante (ayant l'outil WPScan installé).

Télécharger le fichier suivant : docker-compose.yml

Tapez les commandes suivantes :

docker-compose up -d

Pour utiliser la machine attaquante (avec WPScan) tapez :

docker exec -it wpscan bash

3. Lancer le scan avec WPScan

La première à chose à savoir c'est qu'il faut inclure le paramètre "--url" pour spécifier le nom de domaine du site à analyser.

wpscan --url wordpress

Vous voyez s'afficher les informations sur les vulnérabilités de la version de Wordpress.

4. Enumerer les utilisateurs

wpscan --url wordpress -e u

Maintenant que nous avons énuméré les utilisateurs du site, nous pouvons tenter de trouver le mot de passe d’un utilisateur par brute force.
