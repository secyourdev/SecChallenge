Tutoriel : HTTrack

1. Objectif

HTTrack est un aspirateur de site web. Il permet de télécharger un site web internet. Il permet de ne plus avoir besoin d’une connexion pour pouvoir visiter le site.
Le logiciel est téléchargeable sur le site : https://www.httrack.com/

2. Configuration

Pour ce tutoriel, nous utiliserons docker pour avoir une machine et un serveur HTTP apache.

Récupérer le dockerfile dans le lien ci-après : docker-compose.yml
Dans un bash (linux) ou cmd (windows) aller dans le répertoire où vous avez télécharger le fichier docker-compose.yml :
bash> docker-compose up -d

Vous disposez maintenant d’une machine configurée avec HTTrack et un serveur HTTP apache.

3. Les étapes de copie de site web

Une fois le conteneur lancé, connectez-vous à la machine avec la commande :
bash> docker exec -ti tuto-httrack /bin/bash

Vous pouvez vérifier que le répertoire dans httrack dans lequel vous êtes est vide avec la commande :
bash> ls

Pour copier le site web du serveur la commande est :
bash> httrack [-options]
bash> httrack [-option] [+] [-] [+] [-]

Ici nous utilisons l’option -O pour choisir le répertoire dans lequel copié le site web.
bash> httrack http://10.5.0.3 -O ~/httrack


Pour copier seulement les fichiers HTML on utilise l’option –get :

bash> httrack –get http://10.5.0.3 -O ~/httrack2

4. Vérification du téléchargement du site web

Vous pouvez vérifier que le répertoire httrack est bien créer avec :
bash> ls

Puis aller dedans :
bash> cd httrack

Allez ensuite dans le répertoire de l’adresse IP du site que vous avez copié :
bash> cd 10.5.0.3

Vous pouvez vérifier que le fichier index.html du site est bien télécharger :
bash> ls



Ouvrez le fichier avec la commande :
bash> cat index.html

Ou vous pouvez aller dans le répertoire httrack2 :
bash> cd httrack2
bash> ls



bash> cat index-2.html

Ceci conclut ce tutoriel. Pour quitter le docker entrez la commande :
bash> exit

5. Validation

Pour valider le tutoriel, il faut entrer le mot secret contenu dans le fichier index.html ou index-2.html
