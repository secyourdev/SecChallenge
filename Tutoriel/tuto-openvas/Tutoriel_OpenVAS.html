OpenVAS



1. Objectif

L'objectif de ce tutoriel est de présenter l'outil OpenVAS.



2. Introduction

Mais tout d'abord, qu'est-ce qu'OpenVAS ?

Il s'agit d'un outil de scanner de vulnérabilités open-source. Cet outil est utilisé pour identifier et évaluer les vulnérabilités potentielles dans les systèmes informatiques.
Pour cela, OpenVAS se sert d'une variété de techniques de scan (tests de port, tests de service, tests de configuration et tests de vulnérabilités spécifiques) avant de fournir des rapports détaillés sur les problèmes trouvés, avec des recommandations sur les mesures à prendre pour les corriger.



3. Configuration

Téléchargez le fichier suivant : docker-compose.yml

Ouvrez votre invite de commandes

Utilisez les commandes suivantes dans l'ordre donné :
docker-compose up -d
docker pull greenbone/openvas-scanner
docker run greenbone/openvas-scanner

Pour accéder au portail d'OpenVAS, tapez l'URL suivant dans votre navigateur de recherche http://127.0.0.1:9392

Une fois au portail, utilisez les informations suivantes pour vous connecter :
Username : admin
Password : admin



4. Les étapes pour votre premier scan

Bien joué !
Vous êtes arrivé sur Greenbone Security Assistant :


Maintenant mettez votre souris sur "Administration" et cliquez sur "feed status", vous devriez voir les lignes suivantes


Attendez que les "Status" se mettent à jour (s'ils ne le sont pas déjà).
Allez ensuite dans la rubrique "Scans", mettez votre souris sur la page étoilée en haut à gauche et cliquez sur "New Task"


Vous devriez voir apparaître la fenêtre suivante


Renseignez les différentes informations demandées et précisez
Scanner : OpenVAS Default Scan Config : Full and fast

Il s'agit des paramètres les plus souvent utilisés, ils feront largement l'affaire pour votre premier scan.

Sauvegardez la configuration que vous venez de créer et allez dans l'onglet "Tasks" (dans le menu déroulant de la rubrique "Scans") vous devriez en voir apparaître une nouvelle.
Cliquez sur le bouton "Start" en face du test que vous venez de créer pour le lancer.


Attendez que le scan se finisse pour pouvoir voir les résultats et vulnérabilités de l'Host que vous avez scanné dans les onglets "Results" et "Vulnerabilities"



Félicitations, vous avez fait votre premier scan !
