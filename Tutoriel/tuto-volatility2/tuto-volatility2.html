Volatility - Outil pour l'Analyse de Mémoire

Volatility est un outil open-source utilisé pour l'analyse forensique de mémoire. Il permet d'extraire des informations précieuses à partir de l'image mémoire d'un système, telles que les processus en cours d'exécution, les connexions réseau, les fichiers ouverts, et bien plus encore..

Installation de Volatility sur Windows

Pour installer Volatility sur Windows, suivez ces étapes :

  1. Téléchargez et installez Python version 3 au minimum.
  2. Téléchargez et installez GitForWindows.
  3. Téléchargez et installez Microsoft C++ Build Tools.
  4. Téléchargez Snappy (choisissez la version adaptée à votre système et à votre version Python).
  5. Ouvrez un terminal et installez Snappy. pip install chemin-d-acces\python_snappy-xyz.whl (il est important de fournir le chemin d'accès du .whl téléchargé)
  6. Toujours dans le terminal rendez vous là où vous voulez installer volatility et clonez le github de volatilitygit clone https://github.com/volatilityfoundation/volatility3.git puis faites cd .\volatility3\
  7. Il faut maintenant installer les fichiers requis avec pip install -r requirements.txt Volatility est maintenant prêt à l'emploi, vérifiez cela en faisant python vol.py -v

Tutoriel : Visualiser la liste des programmes sur une image mémoire

Dans ce tutoriel, nous allons visualiser la liste des programmes en cours d'exécution sur une image mémoire fournie. La solution finale sera le PID d'une application spécifique.

Étapes :

  1. Téléchargez, extrayez et placez l'image mémoire dans un répertoire accessible depuis votre terminal.
  2. Ouvrez un terminal et accédez au répertoire où vous avez extrait les fichiers de Volatility.
  3. Exécutez la commande suivante pour visualiser la liste des programmes :
    4. python vol.py -f /chemin/vers/l'image_mémoire windows.pslist
  4. Recherchez l'application VBoxService.exe dans la liste affichée et notez son PID.

Solution :

Le PID de l'application VBoxService.exe est 1660.
