Challenge brut-force - Ncrack

1. Objectif

L'objectif de ce challenge est de trouver le mot de passe d'une utilisatrice.
Vous allez attaquez la machine 192.172.20.4, vous n'aurez pas besoin d'attaquer le serveur web. Cette machine à deux utilisateurs dont l'un avec un nom d'utilisateur vraiment générique
Au hasard guest ou user.

2. Installation et Configuration

Si vous avez déjà installer le docker-compose du tutorial passer cette section Ce challenge met à disposition, par le biais de Docker-compose, une machine déjà configurée avec un dictionnaire de mot de passe "rockyoulite.txt". En plus du client ncrack, la liste des machines fournies à qui seront attaqué durant le tutoriel et le challenge.
Récupérez le dockerfile dans le lien suivant, si vous avez déjà installer le docker-compose.yml du challenge, passer à l'étape 3 : docker-compose.yml

Dans un interpréteur de commande,bash (Linux) ou cmd (Windows), allez dans le répertoire où vous avez téléchargé le fichier docker-compose.yml et effectuez la commande suivante :
docker compose up -d
docker exec -it ncrack /bin/bash

Vous disposez maintenant d'une machine ubuntu avec ncrack d'installer. Ainsi que plusieurs autres machines à attaquer. Pour mettre en pause ou arreter les machine faite:
docker compose pause/unpause
docker compose stop/start

Cartographie du réseau :
ncrack-client 192.172.20.2
serveur pure-ftpd 192.172.20.3
serveur ssh 192.172.20.4
serveur web 192.172.20.5
