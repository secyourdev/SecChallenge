Challenge John the Ripper

Pour voir si vous avez bien compris le cours, je vous propose de vous entraîner sur un docker que j’ai créé spécialement pour cela.

Que devez-vous faire ?

Étape 1 : récupérer le docker proposé

Pour cela, il vous faut Docker installé (il existe tout un tas de tutoriels qui vous expliqueront comment le télécharger.) quand Docker et docker-compose sont installés sur votre machine, vous pouvez commencer.
Il faut d'abord télécharger le fichier docker-compose.yml
Ensuite, il vous suffit de taper la commande (en étant dans le bon répertoire où se trouve le fichier que vous venez de télécharger):

docker-compose up -d

Vous allez maintenant acceder au docker attaquant depuis lequel vous tenterez d'utiliser impacket

docker exec -it smb_attacker bash

Situation

Vous êtes connécté sur le même réseaux qu'un serveur smb et un client. Il y a un share réseaux auquel le client accede régulierement.
Vous devez intercepter la connexion entre le client et le serveur afin de récuperer les log vous permettant d'acceder au share dans lequel se situe le flag.
