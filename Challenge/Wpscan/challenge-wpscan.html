Challenge WPScan

Pour voir si vous avez bien compris le cours, je vous propose de vous entraîner sur un docker que j’ai créé spécialement pour cela.

Que devez-vous faire ?

Étape 1 : récupérer le docker proposé

Pour cela, il vous faut Docker installé (il existe tout un tas de tutoriels qui vous expliqueront comment le télécharger.) quand Docker et docker-compose sont installés sur votre machine, vous pouvez commencer.
Il faut d'abord télécharger le dossier docker
Ensuite, il vous suffit de taper la commande (en étant dans le bon répertoire où se trouve le fichier docker-compose.yml):

docker-compose up -d

Étape 2 : lancer la machine

Voilà ! Vous venez de télécharger une machines kali, il faut maintenant la lancer.
Pour utiliser la machine attaquante (avec WPScan) tapez :

docker exec -it wpscan bash
