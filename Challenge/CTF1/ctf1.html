CTF #1

1. Qu'est ce qu'un "CTF"?

Un CTF, ou Capture The Flag, est un type de challenge où le but est de récupérer un flag (très souvent un fichier texte) qui contient une chaine de caractère qui correspond à la réponse finale du challenge.
La difficulté des CTFs est plus élevée que celle des challenges. Il est nécessaire de bien comprendre comment utiliser les différents outils vu dans les différents cours et tutoriels.

2. Description de CTF#1

Vous avez à votre disposition une machine Kali Linux ainsi qu'un docker Firefox sur le port 5801 de votre machine hôte.
Votre but sera de récupérer un flag caché dans une machine. Tous les coups sont permis: vous avez le droit d'installer et utiliser les outils que vous souhaitez afin de réussir ce challenge. Attention à ne pas sortir du cadre du challenge.
Pour accéder à votre machine, télécharger le fichier docker-compose puis dans un terminal, entrez la commande docker-compose up -d --build, puis accéder à votre machine avec la commande docker exec -t ctf1-hacker bash.
